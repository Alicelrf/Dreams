**集故事** **+** **隐线主线** **+** **世界逐渐揭露**

结构大致是：

episodic cases（每集一个故事）
        ↓
character arc（角色关系变化）
        ↓
world mystery（世界观逐渐解锁）
  

**一、病毒宇宙推荐结构比例（****Cowboy Bebop****式）**

如果把整个系列看成 **100%****素材池**，比较稳定的比例是：

|   |   |   |   |
|---|---|---|---|
|**层级**|**内容**|**比例**|**作用**|
|**Case Episodes**|单个梦境/幻觉世界|**45%**|每一集独立故事|
|**Character Arc**|Alice、J、Peter关系|**25%**|情感驱动|
|**World Mystery**|病毒规则与实验|**20%**|世界观逐渐揭露|
|**Poetic Interludes**|诗性场景、梦镜头|**10%**|气质与主题|

- **45%** **单集故事** → 保持阅读节奏
- **25%** **人物关系** → 这是读者真正关心的
- **20%** **世界观** → 太多会变百科全书
- **10%** **诗性段落** → 保持你的风格但不过载
